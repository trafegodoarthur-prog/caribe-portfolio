# Caribé Tráfego Pago — Site de Portfólio

Site de uma página (estático: HTML + CSS + JavaScript puro) da marca
**Caribé Tráfego Pago** — tráfego pago estratégico para autoras
independentes na Amazon (KDP).

## Estrutura

- `index.html` — a página
- `styles.css` — estilos (estética editorial/literária)
- `script.js` — animações, contadores, formulário e o lightbox do "Pódio na Amazon"
- `assets/` — logo e as imagens dos rankings (TOPs da Amazon)

## Desenvolvimento

Não há build nem dependências. Basta abrir `index.html` no navegador
(precisa de internet na primeira vez, para carregar as fontes do Google Fonts).

## Deploy

Hospedado na **Vercel**. Não há configuração: a Vercel serve os arquivos
estáticos a partir da raiz do repositório.
